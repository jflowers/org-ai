# Agent Support and OS Compatibility

This project is designed to support AI Agents and Model Context Protocol (MCP) servers with a focus on specific operating systems.

## OS Compatibility

This repository **only** supports the following operating systems:

- **macOS**: Fully supported.
- **Linux**: Supported distributions include:
  - **Fedora**
  - **CentOS Stream**
  - **RHEL (Red Hat Enterprise Linux)**

> [!IMPORTANT]
> **Windows is NOT supported.** Please do not attempt to use these configurations or tools on Windows environments.

## Agent Environments

The documented configurations are tested and optimized for:
- **Gemini** (via AI Studio and local settings)
- **Cursor** (AI-powered code editor)
- **Claude** (via Claude Desktop)

For detailed setup instructions, please refer to the [MCP Documentation](./mcp-docs/index.md).
