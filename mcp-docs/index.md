# MCP Documentation

This directory contains documentation for configuring and using Model Context Protocol (MCP) servers with various AI clients.

## Structure

- [Servers](./servers/): Configuration steps for specific MCP servers.
  - [GitHub](./servers/github.md)
  - [Chrome](./servers/chrome.md)
  - [Google Workspace](./servers/google-workspace.md)
- [Clients](./clients/): How to connect MCP servers to specific AI tools.
  - [Gemini](./clients/gemini.md)
  - [Cursor](./clients/cursor.md)
  - [Claude](./clients/claude.md)

## Quick Start

1. Find the **Server** you want to set up in the `servers/` directory and follow the prerequisite steps (API keys, permissions).
2. Choose your **Client** (Gemini, Cursor, or Claude) in the `clients/` directory and follow the instructions to connect the server.
